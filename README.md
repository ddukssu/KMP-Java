# KMP String Search

This project contains a simple Java program that uses the KMP algorithm.
The program searches for a pattern inside a text.

## How to run

1. Compile:
   javac KMPSearch.java

2. Run:
   java KMPSearch

The program prints the positions where the pattern appears in the text.

## Files

- KMPSearch.java – Java source code
- REPORT.md – short report about the algorithm
- samples.txt – sample input and output
